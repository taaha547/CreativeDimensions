DOWNLOAD AND HOSTING INSTRUCTIONS

1. DOWNLOADING
- Click the Download ZIP button
- Extract the ZIP file to your computer

2. HOSTING OPTIONS
a) GitHub Pages:
- Create a GitHub repository
- Upload files and enable GitHub Pages

b) Netlify:
- Create Netlify account
- Drag and drop folder to deploy

c) Local Testing:
- Install Node.js
- Run: npx serve
- Visit http://localhost:5000

3. CUSTOMIZATION
- Edit config.js for text and projects
- Place 3D models in /models folder
